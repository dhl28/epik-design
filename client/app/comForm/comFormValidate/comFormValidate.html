formvalidate 表单验证

概述
  • angular表单验证
  • 集成ng-message
实例 验证指令，错误提示
  • 使用angular验证指令
  • 使用ng-message指令设置可重用的错误提示
    • <div ng-message="required">此项为必填项</div>
    • <div ng-message="date">日期格式不正确</div>
    • <div ng-message="fcsaNumber">数字格式不正确</div>
    • <div ng-message="uiSelectRequired">请至少选择一项</div>
    • <div ng-message="minlength">请至少输入10个字符</div>
    • <div ng-message="maxlength">请至多输入100个字符</div>
  • 自定义验证指令的方法

$scope.vm = {}
$scope.datepickerCtrl = {
  input4Opened : false,
  funOpenInput4 : function($event) {
    $event.preventDefault();
    $event.stopPropagation();
    this.input4Opened = true;
  }
}

$scope.classifications = {
